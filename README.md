# Student Enrollment System
a console-based application created while learning Object-Oriented Programming.

## Technologies
- Java
- IntelliJ IDEA
I used Java as a coding language to practice OOP Concepts and IntelliJ because it is one the best IDEs to use Java into.

## Features 
- Student enrollment system
- Object-Oriented Programming
- Aggregation
- do-while loop
- User input using Scanner
